=======================
Contribution guidelines
=======================

* Use `new_frontera google group`_ for all questions and discussions.
* Use `Github repo`_ pull request for submitting patches.
* Use Github repo issues for issues new_frontera could benefit from in the future. Please don't put your own problems running
  new_frontera there, instead use a google group.

We're always happy to accept well-thought solution with documentation and tests.

.. _`new_frontera google group`: https://groups.google.com/a/scrapinghub.com/forum/#!forum/new_frontera
.. _`Github repo`: https://github.com/scrapinghub/new_frontera
